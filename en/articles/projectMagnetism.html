A game made in Unity for of a 2-month university group project. Our goal was to make an action-platformer, a la Metroid, based on the use of magnetic fields in the environment and characters. For example, the playable characted is repelled by magnets of the same polarity, and attracted to the opposite polarities, which forces you to make the right choices when platforming.
I did several different tasks on this project: I did all the gameplay for the main character, all the graphics except for the enemies and particles, the shader used to show the current polarity of the character, the code for the magnets, the code for the camera behaviour, and all the level design.
NB: because of a mistacke while doing back-ups of this project, the latest version is lost, so the repo is not representative of what's shown in the video. The slides provided are mainly there because of the commentaries still present, which explain our intentions behind the project very well.
